# 🔄 快速切换到本地网络

## ⚠️ Sepolia 当前问题

你遇到的错误：
```
Relayer didn't response correctly
Transaction rejected: Input request failed
backend connection task has stopped
```

**原因**：Zama 的 Sepolia Relayer 服务可能暂时不可用或有问题。

**建议**：先切换到本地 Hardhat 网络进行开发和测试。

---

## 🚀 3 步切换到本地网络

### 步骤 1：在 MetaMask 切换网络

1. **打开 MetaMask**
2. **点击顶部网络名称**（当前显示 "Sepolia Test Network"）
3. **选择 "Hardhat Local"** 或 **"Localhost 31337"**

**如果没有这个网络**，手动添加：
```
网络名称: Hardhat Local
RPC URL: http://localhost:8545
Chain ID: 31337
货币符号: ETH
```

### 步骤 2：刷新页面

按 `Ctrl + F5` 硬刷新浏览器

### 步骤 3：验证

检查页面显示：
```
FHEVM STATUS
Ready ✅
chainId 31337  ← 应该是 31337，不是 11155111
```
